📥 Download 美业素材库

这个 folder 是 "Anyone with link can view", 但是 Drive 2024+ 不允许未登录 download. 你需要选一种方法:

方法 A · 浏览器手动下载 (5 分钟, 最简单)

  1. 打开 美业素材库 (你必须已登录 Google)
  2. 点 folder 上面的 ⋮ 三个点 (在 folder 名旁边)
  3. 选 "Download" → 会下个 zip `美业素材库.zip`
  4. 把 zip 移到 `~/Downloads/`
  5. 告诉我 zip 路径, 我解压 + 上传

方法 B · 建 Service Account (10 分钟, 永久方案)

  1. 打开 Google Cloud Console
  2. 新建一个 project (e.g. "gobeauty-drive")
  3. 左侧 "APIs & Services" → "Enable APIs" → 搜 "Google Drive API" → Enable
  4. "APIs & Services" → "Credentials" → "Create Credentials" → "Service Account"
  5. Name: gobeauty-reader, Role: 不需要, Done
  6. 点新建的 service account → "Keys" tab → "Add Key" → "Create new" → JSON → download
  7. JSON 文件移到 `~/secrets/drive-sa.json`
  8. 回到 Drive folder, share 给 service account email (在 JSON 文件里 `client_email` 字段, 形如 `gobeauty-reader@xxx.iam.gserviceaccount.com`)
  9. 把 service account email 发给我, 我写脚本拉

方法 C · 你 share 给我 (2 分钟, 一次性的)

  1. 打开 美业素材库
  2. 点 "Share" → 改成 "Anyone with link" + "Viewer"
  3. 然后... (其实你已经设过了, 但还是 500)
  4. 算了, 这个方法不行, 跳过

推荐 A (5 分钟), 你 zip 下来后回我 "下载了", 我立刻处理.
